Le SE-214 est un boîtier robuste certifié IP67. Les caractéristiques d'étanchéité et d'anti-poussière permettent une utilisation dans les applications extérieures. Ce produit peut être personnalisé par usinage CNC, impression UV et marquage laser. Le produit est fabriqué en ABS. Un couvercle transparent en PC est disponible en option.
